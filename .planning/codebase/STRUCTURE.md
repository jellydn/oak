## Naming Conventions

| Convention | Example |
| --- | --- |
| ViewModel: `*ViewModel` | `FocusSessionViewModel` |
| Service: `*Manager` or `*Service` or `*Store` | `AudioManager`, `NotificationService`, `PresetSettingsStore` |
| Tests: `*Tests.swift` | `US001Tests.swift`, `NotchCompanionViewTests.swift` |
| Extensions: `Type+Feature.swift` | `NotchCompanionView+Controls.swift`, `NSScreen+UUID.swift` |
| User story tests: `US00XTests.swift` | `US001Tests.swift` through `US006Tests.swift` |
| Protocol: `*ing` or `*Protocol` | `SessionCompletionNotifying`, `AudioEngineProtocol` |
| Mocks: `Mock*` | `MockAudioManager`, `MockNotificationService` |
