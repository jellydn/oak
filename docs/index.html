üöÄ Getting Started

Using Homebrew (Recommended)

# Add the tap
brew tap jellydn/oak https://github.com/jellydn/oak

# Install Oak
brew install --cask oak

From Source

Prerequisites:

  • macOS 13+ (Apple Silicon recommended)
  • XcodeGen (brew install xcodegen)
  • SwiftLint (optional: brew install swiftlint)
  • SwiftFormat (optional: brew install swiftformat)
# Clone the repository
git clone https://github.com/jellydn/oak.git
cd oak

# Generate Xcode project
cd Oak && xcodegen generate

# Build and run
open Oak.xcodeproj
